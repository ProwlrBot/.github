## The Ecosystem

Security-first tools for bug bounty, red team, and agent-ops. Every repo here is something we actually use.

| Repo | What it is |
|---|---|
| [**prowlrbot**](https://github.com/ProwlrBot/prowlrbot) | The platform — multi-agent coordinator, war-room, mission runner |
| [**CyberBox**](https://github.com/ProwlrBot/CyberBox) | All-in-one Docker security workspace — 160+ tools, dual AI, Caido proxy |
| [**prowlr-memory**](https://github.com/ProwlrBot/prowlr-memory) | Local RAG over bounty notes, session transcripts, and program scopes |
| [**prowlr-doctor**](https://github.com/ProwlrBot/prowlr-doctor) | Security-aware Claude Code environment auditor with TUI |
| [**roar-protocol**](https://github.com/ProwlrBot/roar-protocol) | Real-time Open Agent Runtime — MCP + A2A + ACP + ANP in one spec |
| [**prowlr-marketplace**](https://github.com/ProwlrBot/prowlr-marketplace) | Registry for skills, agents, prompts, MCP servers, themes, workflows |
| [**prowlr-docs**](https://github.com/ProwlrBot/prowlr-docs) | Official documentation |

## Quick start

```bash
pip install prowlrbot
prowlr init
prowlr app
```

Full install: [prowlrbot.com](https://prowlrbot.com) · Docs: [prowlr-docs](https://github.com/ProwlrBot/prowlr-docs)
