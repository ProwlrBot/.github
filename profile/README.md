### What is ProwlrBot?

**ProwlrBot** is an autonomous AI agent platform where agents don't just respond &mdash; they **hunt**, **coordinate**, and **evolve**. Build packs of specialized agents that work together in real-time, powered by a flexible protocol stack (MCP + ROAR + ACP).

```bash
pip install prowlrbot
prowlr init --defaults
prowlr app
```

> *"The impediment to action advances action. What stands in the way becomes the way."*
> &mdash; Marcus Aurelius

</td>
<td width="50%" valign="top">

| | Others | ProwlrBot |

## Quick Start (60 seconds)

```bash
# Install the platform
pip install prowlrbot

# Initialize with smart defaults
prowlr init --defaults

# Launch your first agent
prowlr app

# Activate the war room for multi-agent coordination
prowlr hub start
prowlr team create "my-pack"
```
